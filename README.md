# Useful KiCad Hotkeys (5.1)

## EEschema

Can also be found under Help -> List Hotkeys 

### Keyboard
HOME -  Zoom to Fit 
A	 -  Add Symbol
C    -  Duplicate Symbol
R	 -  Rotate Item
G    -  Move Item
W	 -  Begin Wire

### Mouse

Middle Mouse Button Push - PAN
Scroll					 - Zoom
